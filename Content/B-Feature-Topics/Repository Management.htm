Feature 1

Repository Management

Previously, the feature was only available in the Desktop Client and provided repository management for

  • Alert History (IAlertHistory)

  • CI Scan History (IManagedResourceDetailDiff)

  • Admin log (IAdminLog)

  • User log (IUserLog)

In this release, we enhanced Repository Management to include History Actions and their sub-records as part of ChangeGear regular maintenance and made this feature available in Web Admin.
