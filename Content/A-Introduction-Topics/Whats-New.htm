ChangeGear Web Admin 9.0

Release Overview

Release Date: Tuesday, December 13, 2022

Version: 9.0

Change Summary: This release includes multiple enhancements and fixed issues. ChangeGear Web Admin was enhanced to make the repository management feature available. Repository Management will now include History Actions and their subrecords as part of ChangeGear regular maintenance.

Repository Management

Previously, the feature was only available in the Desktop Client and provided repository management for

  • Alert History (IAlertHistory)

  • CI Scan History (IManagedResourceDetailDiff)

  • Admin log (IAdminLog)

  • User log (IUserLog)

In this release, we enhanced Repository Management to include History Actions and their subrecords as part of ChangeGear regular maintenance and made this feature available in Web Admin.
