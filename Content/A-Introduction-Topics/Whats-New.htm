ChangeGear Web Admin 9.0

Release Overview

Release Date: Tuesday, December 13, 2022

Version: 9.0

Change Summary: ChangeGear Web Admin was enhanced to make the repository maintenance feature available and enhance it to include IncidentRequestHistoryRecords/SubActionHistoryRecords etc. as part of ChangeGear regular maintenance.

Repository Management

Previously, the feature was only available in the Desktop Client and provided repository maintenance for

  • Alert History (IAlertHistory)

  • CI Scan History (IManagedResourceDetailDiff)

  • Admin log (IAdminLog)

  • User log (IUserLog)

In this release, we enhanced repository maintenance to include IncidentRequestHistoryRecords/SubActionHistoryRecords etc. as part of ChangeGear regular maintenance and made this feature available in Web Admin.
